# StyleCurate – Fashion Recommendation Web App

StyleCurate is a web-based application designed to help users discover outfit ideas based on their preferences and style choices.

## Live Demo
https://style-curate-vibes.lovable.app

## Features
- Outfit suggestion system (prototype)
- Clean and modern user interface
- Preference-based recommendation concept

## How It Works
1. User selects style or occasion
2. Inputs preferences (color, vibe, etc.)
3. System suggests outfit ideas

## Tech Used
- Web-based development platform
- AI-assisted development tools

## Status
Prototype (UI implemented, logic in progress)
